基于 pnpm 搭建的 monorepo 架构的知识库。主要用于归纳 vue3 生态的解决方案。希望各位小伙伴一起讨论总结，一起进步。后期计划加入 react&nextjs 等子项目。看个人时间。

### 初始化依赖

```
pnpm install
```

### Lints 检测代码规范与修复

```
pnpm lint:lint-staged
```
